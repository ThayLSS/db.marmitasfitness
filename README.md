# 🍽️ API - Marmitas Fitness

Esta é a API que gerencia o cardápio, os tamanhos das marmitas, as bebidas e o status dos pedidos para o projeto **Marmitas Fitness**.


## 📌 Rotas Disponíveis

A API possui os seguintes endpoints principais para consumo dos dados:

* **Marmitas (Cardápio):** `GET /marmitas`
* **Pedidos Realizados:** `GET /pedidos` ou `POST /pedidos`
* **Bebidas:** `GET /bebidas`
* **Status dos Pedidos:** `GET /status_pedido`
* **Tamanhos:** `GET /tamanhos_marmita`
